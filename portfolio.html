image-editor

An image editor written in TypeScript that applies effects to PPM image files.
Supported filters: motionblur, invert, emboss, grayscale.
Usage:

  • To convert any image to PPM format, open the image in GIMP, select "Export As", then change the file extension to PPM. Click "Export", then select "ASCII" (otherwise it won't work). Then, go into the image file, and remove the comment so only the P3 header and numbers remain.
  • From ./src, run: ts-node imageeditor.ts <imagePath> <sameImagePath> <filterType>
  • Reopen the modified image in GIMP to view results.
